Editor

Hotkey Action
Ctrl + k Toggles comment for the current line of the cursor or the highlighted lines.
Ctrl + space Provides instruction or register suggestions. Will also complete the instruction text if there is a single possibility.
Button Action
Open Opens and reads the contents of a file from your computer to the editor.
Save Saves the text in the editor to your computer as an ASM file.
Run Runs the program currently loaded in main_memory.

Dropdown options:

    Run (ignore breakpoints) - runs loaded program while ignoring breakpoints. Load and Run - assembles and loads the program and runs loaded program. Load and Run(ignore breakpoints) - assembles and loads program and runs loaded program while ignoring breakpoints.
Pause Pauses a running program.
Step Executes only the current instruction being pointed to by the program counter.
Refresh Resets all data registers to 0, SP to the MAX_ADDRESS, PC to 0, and zeroes out main memory.
Load Assembles and loads program in editor and loads into main memory.

Register Table

Option Action
PC Jumps to the program counter in memory.
SP Jumps to the stack pointer in memory.
Decimal/Hex This drop down menu toggles the registers and main memory to the selected base.

Main Memory

Option Action
Speed Toggles the speed at which to run the loaded assembly program. No delay does not mean the program will run instantly but instead means there is no artificially inserted delay.

Console

Button Action
Clear Console Clears all messages in the console.
